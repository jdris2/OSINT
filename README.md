# OSINT Intel Engine

A comprehensive intelligence gathering and analysis system for OSINT research.

## Project Structure

- **docs/** - Project documentation and guides
- **intel_engine/** - Main package
  - **core/** - Base module classes
  - **ai/** - AI planning and orchestration
  - **modules/** - Research modules (ABN lookup, Twitter activity, etc.)
  - **schemas/** - Data schemas and configurations
  - **tests/** - Test suite
  - **reports/** - Report generation

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python run.py
```

## Documentation

See the `docs/` directory for detailed guides and system documentation.
